## Long Description (4000 characters max)

**Current Length: 1,247 characters**

Ampush Motor Controller is an application for remotely controlling and monitoring IoT motor devices through SMS communication.

FEATURES

Motor Control
- Send motor control commands via SMS (MOTORON, MOTOROFF, STATUS)
- Real-time motor status monitoring
- View voltage, current, and water level readings
- Monitor motor runtime and operating mode

Device Management
- Manage multiple IoT motor devices
- View device information and last activity
- Track device status and connectivity

Data Synchronization
- Automatic synchronization of motor operation logs
- Local data storage with cloud backup
- View historical motor operation data
- Generate reports and analytics

Notifications
- Receive push notifications for motor status changes
- Get alerts for important events
- Real-time updates on motor operations

User Account
- Secure account management
- Profile customization
- Multiple device support
- Data privacy controls

REQUIRED PERMISSIONS

SMS Permissions
- SEND_SMS: Required to send motor control commands
- RECEIVE_SMS: Required to receive motor status updates
- READ_SMS: Required to process motor-related SMS messages

Phone Permissions
- READ_PHONE_STATE: Required to detect phone number for device identification
- READ_PHONE_NUMBERS: Required for device registration

Other Permissions
- INTERNET: Required for data synchronization with backend servers
- POST_NOTIFICATIONS: Required for push notifications
- Background service permissions for data sync

Note: The app only accesses SMS messages directly related to motor control operations. Other SMS messages on your device are not accessed, read, or transmitted.

DATA COLLECTION

The app collects the following information:
- Account information (name, email, phone number)
- Motor operation data (status, voltage, current, water level)
- Device information (device name, SMS number, activity timestamps)
- Technical information (app version, device model, FCM tokens)

All data is stored securely and used only for providing motor control services. For detailed information, please review our Privacy Policy available in the app.
